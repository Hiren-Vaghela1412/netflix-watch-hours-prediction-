# Netflix Watch Hours Prediction

## Project Overview

This project focuses on analyzing Netflix content data and predicting watch hours using Machine Learning techniques.

The workflow includes:

* Data Cleaning
* Missing Value Handling
* Exploratory Data Analysis (EDA)
* Outlier Detection and Treatment
* Feature Engineering
* Data Preprocessing using Pipelines
* Model Training and Evaluation
* Performance Comparison of Multiple Regression Models

## Dataset Features

The dataset contains information related to Netflix content, including:

* Watch Hours (Millions)
* Marketing Spend
* Content Attributes
* Numerical and Categorical Features

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Workflow

### 1. Data Preprocessing

* Missing value imputation
* Numerical feature scaling
* Categorical feature encoding
* ColumnTransformer and Pipeline implementation

### 2. Exploratory Data Analysis

* Distribution Analysis
* Boxplots for Outlier Detection
* Correlation Heatmap
* Scatter Plots
* Feature Relationships

### 3. Model Building

Multiple regression models were trained and evaluated using:

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

## Results

The models were compared based on their predictive performance, and the best-performing model was selected for watch-hours prediction.
